## 2026-04-25 Task 3: NarrativeState Definition
- LangGraph.js Annotation API: `Annotation.Root({ fieldName: Annotation<Type>({ reducer, default }) })`
- For last-writer-wins: use `reducer: (_prev, next) => next` with a `default` function
- For concat: use `reducer: (prev, next) => prev.concat(next)` with `default: () => []`
- For messages field: use `messagesStateReducer` from `@langchain/langgraph` (handles ID-based merging, not just blind concat)
- Type derivation: `type NarrativeState = typeof NarrativeStateAnnotation.State`
- The `Annotation` class is imported from `@langchain/langgraph`, not a separate package
- Next agent type union: `"gm" | "actor" | "scribe" | "archivist" | "end"` — the `"end"` value is a routing signal, not LangGraph's END constant
